Result Scope

This page publishes the current combine result for mStop=1250, mLSP=500 using the corrected Z→νν tail in the nominal template and the internal shape systematics available in this partial production snapshot.

This is not yet a final full-production result. The original full preview missed old-MC Z→νν tail shards after shard_oldmc_01966; this page applies a direct correction from the readable tail payloads. Fifteen tail payloads still failed EOS reads/timeouts and are listed in zinv_tail_correction.json.
