## Changes: 2026-05-06 (Scale factors for 2025)

Merge Request: [!3](https://gitlab.cern.ch/cms-analysis-corrections/EGM/Run3-25Prompt-Summer24-NanoAODv15/-/merge_requests/3)

Release of scale factors for 2025 data taking period. 
These scale factors must be applied on MC campaign: **Summer24** (including the 2025 puWeight).

The included SF are: 
- electron reco SF for pT > 20 GeV
- electron ID SF for cut based and MVA based ID 
- photon ID SF for cut based and MVA based ID

Missing SF (will be added in a next release): 
- reco SF pT < 20 GeV 
- electron ID promptMVA
- trigger SF
